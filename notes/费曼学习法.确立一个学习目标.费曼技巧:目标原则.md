# 一个目标需要符合五个原则，才是值得投入精力为之努力的目标

- 目标的全面性原则
>- 制定的目标要有侧重点
>- 制定的目标也要有针对性（针对自己的某项不足）

- 目标的挑战性原则
>- 制定的目标要有挑战性
>- 制定的目标要能挖掘和激发自己的潜能
>- 制定的目标不能在学习的过程中认为的调低难度

- 目标的可行性原则
>- 制定的目标要切实可行
>- 制定的目标必须符合我们的客观实际

- 目标的可调性原则
>- 可调性就是随着环境和内外条 件的变化，我们能对学习的目标进行必要的调整，适应变化。
>- 制定的目标要在实施过程中留有余地。即，准备好多种学习方案 和备用计划。
